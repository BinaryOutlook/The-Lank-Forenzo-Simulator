The Lank Forenzo Simulator / V0.4

HMAS readiness before the next offensive

V0.4 converts the current playable build into a command-ready architecture. The player loop stays intact while contracts, persistence, simulation boundaries, content diagnostics, and quality gates become clear enough for five platoons to operate in parallel.

Mission Character

This is not a feature spectacle. It is a discipline pass: split the main simulation command post, define shared contracts, make saves migratable, expose content risks, and give QA enough seeded data to detect regressions before the game grows again.
