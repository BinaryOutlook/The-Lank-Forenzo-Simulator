# V0.8 PRD: Explicit Round Flow

Status: implemented for issue #69  
Owner: BinaryOutlook  
Last updated: 2026-05-12  
Iteration type: Round UX, state structure, and fitted run layout

## Summary

`V0.8` restructures the active run screen around an explicit three-phase round loop: **Read / Board Packet → Choose Plays → Resolve / End Round**. The simulation rules, decision selection limits, deterministic resolution path, and authored content stay unchanged; the change is a player-facing state and layout structure that gives each job its own surface.

## Goals

1. Make the move from situational reading into decision selection obvious and intentional.
2. Give the decision tray a dedicated choice phase instead of forcing it to compete with every surrounding run panel.
3. Move end-round confirmation into a resolve phase tied to selection and reserve validation.
4. Preserve fitted, no-document-scroll browser play across the supported viewport projects.
5. Keep the severe, ledger-like, institutional interface tone already established by the app shell and themes.
6. Preserve existing app navigation, including Run, About, Tutorial, and Options.

## Non-Goals

- Do not change simulation outcomes, seeded determinism, content packs, decision impacts, or ending rules.
- Do not add new routes for the round phases.
- Do not replace the Tutorial, About, Options, or landing flows.
- Do not introduce new dependencies or a separate mobile gameplay mode.

## Implementation Notes

- Round flow state is local to `RunScreen` and resets to the read phase when the round number changes.
- The resolve phase is gated until the player has entered the choose phase for the current round.
- Holding the line with zero selected plays remains a valid resolution path.
- The `QuarterControls` component now supports disabled and helper states so the resolve button can reflect phase and selection validity.
- Pure round-flow helpers live in `src/screens/run/roundFlow.ts` with unit coverage.
- Playwright smoke coverage now drives the active run through the three explicit phases.
