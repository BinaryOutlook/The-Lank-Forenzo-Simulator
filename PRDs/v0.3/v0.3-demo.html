V0.3 demo reference

A deeper tray, nastier playbooks, better round-to-round texture.

This reference captures the direction for `V0.3`: the decision tray grows from a small starter set into a multi-pack library of takeover grooming, creditor warfare, shell-carrier arbitrage, safety denial, asset harvesting, and executive escape plans. The content scale matters, but so does curation. A bigger tray only wins if the round still feels edited and strategically varied.

`V0.3` should treat the current decision library as a floor, not a destination. The new tray must feel like a real catalogue of airline-stripper tactics rather than five recurring flavors of austerity.

Curation rules

The expansion only works if the tray avoids obvious sameness. Content scale and curation must ship together.

Prefer at least 3 distinct tactic clusters in a round Suppress exact immediate repeats when the eligible pool allows it Do not let generic high-score cards bury flagged follow-ups forever Keep exit cards special without crowding out the main tray
  • The bigger library should be organized in packs so future decision writing does not collapse back into one giant file.
  • Historical inspiration should be visible as tactic flavor, not as named real-person cosplay.
  • Every aggressive playbook family should have enough delayed fallout to keep the tray morally radioactive.
